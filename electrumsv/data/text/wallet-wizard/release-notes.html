Changes included in this version of ElectrumSV are documented in a more readable form within the following article: ElectrumSV 1.3.0

The most prominent changes are also summarised below for convenience, if you wish to read more about any given change, please read the linked article.

  • The wallet storage has changed from a JSON file to an Sqlite database.
  • A wallet can now have multiple accounts, although they are currently limited to having only one account until the user interface is reworked to support more than that.
